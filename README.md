# Fitness Website Project

This is a multi-page website built using HTML, CSS, and JavaScript.

## Features
- Shopping cart using sessionStorage
- Add, view, and clear items in cart
- Contact form that stores user input using localStorage
- Responsive layout and clean UI

## Technologies Used
- HTML
- CSS
- JavaScript
- Web Storage API (localStorage & sessionStorage)
