# Fitness Website Project

This project is a multi-page website built using HTML, CSS, and JavaScript. It includes interactive features such as a shopping cart and a contact form using browser storage.

## 🚀 Features
- Add items to cart using sessionStorage
- View and clear cart items dynamically
- Process orders with cart reset functionality
- Contact form that stores user data using localStorage
- Responsive and user-friendly design

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Web Storage API (localStorage & sessionStorage)
